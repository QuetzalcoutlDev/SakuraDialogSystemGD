<h2 align = "center"> Sakura Dialog System for GDevelop </h2>
<h6 align = "center"> 🗨️ A dialogue system built from scratch that aims to be more modern, for GDevelop</h6>

![version-badge]  [![license-badge]][license] [![gdevelop-badge]][gdevelop] [![kofi-badge]](https://ko-fi.com/quetzalcoutl)

A dialogue system created from scratch to facilitate the creation of narrative games or visual novels with GDevelop.

> [!NOTE]
> This extension is still **experimental**; it may have bugs and lack some features.

To report a bug, [click here!](https://github.com/QuetzalcoutlDev/SakuraDialogSystemGD/issues)

## Why Sakura Dialog System?

This dialog system arose from observing the limitations of GDevelop's internal dialog system (DialogueTree//Yarn).

Extending the functionality of GDevelop's DialogueTree//Yarn proved very difficult, and after much struggle, the same errors kept appearing. I decided to create my own system from scratch, tailored to my needs as a former Ren'Py user.

This extension also led to its own language, Sakura Script.

Inspired by Ren'Py's own language.

*The name "Sakura" simply appealed to me, and the character featured in the logo is a succubus named "Kammi Sakura."*

For more information about Sakura Script [click here](https://github.com/QuetzalcoutlDev/sakura-script)
